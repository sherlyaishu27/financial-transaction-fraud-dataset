New Measures – Sheet1

1. Total Transactions
Total Transactions = COUNT(Sheet1[type])

2. Total Transaction Amount
Total Transaction Amount = SUM(Sheet1[amount])

3. Average Transaction Amount
Average Transaction Amount = AVERAGE(Sheet1[amount])

4. Total Fraud Transactions
Total Fraud Transactions = SUM(Sheet1[isFraud])

5. Fraud Rate
Fraud Rate = DIVIDE([Total Fraud Transactions], [Total Transactions]) * 100
New Table 1
Fraud Transactions Table
Fraud_Transactions = FILTER(Sheet1, Sheet1[isFraud] = 1)
New Measure → Fraud Table
Fraud Count = COUNT(Fraud_Transactions[type])
New Table 2
Non-Fraud Transactions
NonFraud_Transactions = FILTER(Sheet1, Sheet1[isFraud] = 0)
New Measure
NonFraud Count = COUNT(NonFraud_Transactions[type])
